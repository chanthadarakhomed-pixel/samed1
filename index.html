តើ Game Mobil legend ជាធ្វី?

Mobil legend ជា Game មួយដែលបានបង្កើតឡើងដំបូង ដោយប្រទេសចិនក្នុងឆ្នាំ 2016 ត្រូវបានទិញផ្តាច់មុខដោយក្រុមហ៊ុន Moonton បានដាក់ដំណើរការអោយ លេងជា Globle ក្នុងឆ្នាំ 2017 មានមនុស្សលេងទូទាំងពិភពលោកនិងមានការរៀបចំប្រគួតតាមប្រទេស នីមួយដែលហៅថា MPL

តើ វត្តមានរបស់ Game mobil Legend បានជួយអ្វីខ្លះដល់វិស័យ Esport Cambodia?

វត្តមានរបស់ Game mobil legend បានជួយដល់វិស័យ Esport កម្ពុជាអោយមានការអភិវឌ្ឍជួយដល់ក្មេងៗដែលមានសម្ថភាពកុំអោយលេង game អត់ប្រយោជន៏អោយបានចូលរួមប្រគួតបង្ហាញសម្ថភាពអោយក្រុមធំៗចាប់អារម្មណ៏នឹងយកទៅលេងនៅ MPL
